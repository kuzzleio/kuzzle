# getConfig



Returns the current Kuzzle configuration.

This route should only be accessible to administrators, as it might return sensitive information about the backend.

---

## Query Syntax

### HTTP

```http
URL: http://kuzzle:7512/_getConfig
Method: GET
```

### Other protocols

```js
{
  "controller": "server",
  "action": "getConfig"
}
```

---
