# setex



Sets a value and a time to live (in seconds) on a key. If the key already exists, it is overwritten.

[[_Redis documentation_]](https://redis.io/commands/setex)

---

## Query Syntax

### HTTP

```http
URL: http://kuzzle:7512/ms/_setex/<_id>
Method: POST
Body:
```

```js
{
  "value": "<value>",
  "seconds": 60
}
```

### Other protocols

```js
{
  "controller": "ms",
  "action": "setex",
  "_id": "<key>",
  "body": {
    "value": "<value>",
    "seconds": 60
  }
}
```

---

## Argument

- `_id`: key identifier

---

## Body properties

- `seconds`: expiration duration, in seconds
- `value`: new key value

---

## Response

Returns an acknowledgement.

```js
{
  "requestId": "<unique request identifier>",
  "status": 200,
  "error": null,
  "controller": "ms",
  "action": "setex",
  "collection": null,
  "index": null,
  "result": "OK"
}
```
