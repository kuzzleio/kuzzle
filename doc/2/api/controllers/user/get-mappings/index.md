# getMappings

<SinceBadge version="auto-version"/>

Gets the mappings of the internal users collection.

---

## Query Syntax

### HTTP

```http
URL: http://kuzzle:7512/user/_mappings
Method: GET
```

### Other protocols

```js
{
  "controller": "user",
  "action": "getMappings"
}
```

---
