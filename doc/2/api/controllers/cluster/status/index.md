# status

<SinceBadge version="auto-version"/>

Gets the current cluster status.

## Query Syntax

### HTTP

```http
URL: http://kuzzle:7512/_cluster/status
Method: GET
```

### Other protocols

```js
{
  "controller": "cluster",
  "action": "status"
}
```
